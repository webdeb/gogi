Powerful Features

Natural Language to Shell

Tell Gogi what you want to do and it will propose the correct terminal command. No more searching StackOverflow for complex flags.

Safe Execution Loop

Gogi intercepts proposed commands and explicitly asks for your permission before executing anything. It captures stdout/stderr to learn from mistakes.

Multi-Provider OAuth

Sign in securely using native Device-Code OAuth flows. Plug Gogi directly into your existing OpenAI, Google Gemini, or GitHub Copilot accounts for free!

Installation

Install the package globally using npm to use the `gogi` command from anywhere.

Terminal
# Install globally via npm
npm install -g @webdeb/gogi

Authentication

Before using Gogi, authenticate with your preferred provider. Gogi handles the OAuth flow via your browser and securely stores the token.

Terminal
# Login with ChatGPT Plus (Default)
gogi login

# Or specify a different provider:
gogi login gemini
gogi login github

# Easily switch between active sessions:
gogi provider gemini
